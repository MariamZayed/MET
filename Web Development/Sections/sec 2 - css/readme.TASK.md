# Task: Styling the Academic Portfolio with External CSS

## Objective

Create an external CSS file (`style.css`) to style the HTML portfolio page using modern **CSS3 features**.

You must **not use inline styles or internal styles**. All styling must be written in the external CSS file.

---

# Requirements

## 1. General Page Styling

Apply the following styles to the whole page:

* Set a **background color or gradient** for the page.
* Change the **default font family**.
* Set **margin and padding reset**.
* Center the main content using **width and margin auto**.

Properties you should use:

* `font-family` 
    - choose whatever fonts you like
* `background-color`
    - use neutral color
* `line-height`
    - make it 1.6

---

## 2. Header Styling

Style the name and title section.

Requirements:

* Make the `<h1>` larger and more bold.
* Change the color of the title.
* Add **text-shadow** to the main heading.
* Add spacing between letters.

Properties to use:

* `font-size`
* `color`
* `text-shadow`
* `margin`
    - make it margin-top: 30px
* `letter-spacing`
    - make it 2px

---

## 3. Navigation Bar Styling

Convert the `<ul>` navigation into a horizontal menu.

Requirements:

* Remove default list styling.
* Display menu items horizontally.
* Add different background color to navigation.
* Add hover effect.

Properties to use:

* `list-style`
    - list-style:none;
* `display`
    - ul li { display: inline-block}
* `padding`
* `margin`
* `text-decoration`
* `:hover`

---

## 4. Image Styling

Style the profile image.

Requirements:

* Make the image **circular**.
* Add a **border**.
* Add **shadow**.
* Center the image.

Properties to use:

* `border-radius`
* `border`
* `box-shadow`
* `display`
* `margin`

---

## 5. About Section

Improve the readability of the paragraph.

Requirements:

* Increase line spacing.
* Limit text width.
* Justify the text.

Properties to use:

* `line-height`
* `max-width`
* `text-align`

---

## 6. Contact Form Styling

Style the form fields.

Requirements:

* Inputs should have padding.
* Add border and rounded corners.
* Style the submit button differently.
* Add hover effect to the button.

Properties to use:

* `padding`
* `border`
* `border-radius`
* `background`
* `cursor`
* `:hover`

---

## 7. CSS3 Effects

Use at least **three of the following CSS3 features**:

* `transition`
* `transform`
* `box-shadow`
* `text-shadow`
* `border-radius`
* `linear-gradient`

Example:

* Smooth hover animation on buttons or links.

---

## 8. Layout Improvement

Use **one modern layout technique**:

Choose **one**:

* `Flexbox`
* `Grid`

Example uses:

* Centering elements
* Organizing sections

---

# Submission Requirements

Students must submit:

1. `index.html`
2. `style.css`

The CSS file must include:

* At least **25 CSS rules**
* At least **5 CSS3 features**
* External CSS only
